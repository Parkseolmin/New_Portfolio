안녕하세요, 신입 프론트엔드 개발자 박설민입니다.
프론트엔드 개발은 상상하는 것을 현실로 만들어내는 매력적인 분야라고 생각합니다. 자신의 아이디어를 실제로 구현하기 위해서는 노력과 역량이 필요하며, 저는 아직 많이 배워야 하고 더욱 노력해야 한다는 것을 인지하고 있습니다. 이러한 노력을 위해 도전하고, 성장하는 과정 속에서 얻는 성취감은 프론트엔드 개발을 선택한 이유입니다. 앞으로도 끊임없이 배우고 발전하여 최고의 프론트엔드 개발자가 되도록 노력하겠습니다.

"과정에서 재미를 느끼지 못 하는데, 성공하는 일은 거의 없다." 말 그대로 한 분야에서 성공을 하거나, 더 열정적으로 하기 위해서는 하는 일에 즐거움을 느낄 수 있어야 한다고 생각합니다. 그것이 저에게 코딩이라는 일입니다.

"능력이 없는 것을 두려워 말고, 끈기 없는 것을 두려워 하라." 끊임없이 변화하는 시대에 맞춰 가기 위해 계속해서 열심히 공부하고 있으며, 개발자의 무한한 세계를 알게 되면서 계속해서 발전하며 성장할 수 있는 저의 모습을 볼 수 있었습니다. 앞으로도 꾸준히 자기개발을 하며 발전하는 모습을 보여드리겠습니다.

무엇을 배웠는가 ?

1. HTTP Request Method

HTTP 요청 메서드를 사용하여 데이터 요청, Web App은 JavaScript를 사용하여 서버에 HTTP 요청을 보내며, 이때 주로 Fetch API나 Axios와 같은 라이브러리를 사용하여 간편하게 HTTP 요청을 생성함.

2. JavaScript의 구성요소

Stack, Heap, Task Queue, Event Loop, 웹 api

웹 api : Ajax요청, setTimeout(), 이벤트 핸들러의 등록과 같이 웹 브라우저에서 제공하는 기능을 하며, 쓰레드에서 작업이 이루어지는 것이 아닌 별도의 쓰레드에서 이루어짐

3. async / await

동기적인 자바스크립트를 비동기적으로 처리하기 위해 사용. fetch는 promise를 리턴함. promise 속에서 api호출이 이루져 데이터를 받은 다음 promise가 성공적으로 resolve 함수를 호출하기를 기다리는 것을 await이 해줌

3. Json

서버 클라이언트 통신에서 많이 쓰이는 데이터 타입이며, 객체랑 같아 읽어오기가 편하여 서버통신에서 많이 사용되며 fetch를 쓸 때 항상 사용함.
